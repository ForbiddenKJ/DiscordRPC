# DiscordRPC
A GUI That allows you to customise your Discord Rich Presences, Currently only for linux.
# Linux

__Compiling__
- `pip install pyinstaller`
- `pip install pypresence`
- `pyinstaller discordrpc.py -F`
- `./dist/discordrpc`

__Python__
- `pip install pypresence`
- `python discordrpc.py`
